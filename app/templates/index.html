Origin a self-replicating GitHub repository

This application and its source code together form a self-replicating GitHub repository.

Application is created primarily for education purpose. Functionality is dead simple: it allows you to sign in to your GitHub account and clone its source code ("fork" in GitHub terms). You get a new repository with full application source code copy, licensed under MIT. You can then run your own application instance following the installation instructions on app's homepage on GitHub.

To fork the repository, please login to your GitHub account and then click on the button below.
